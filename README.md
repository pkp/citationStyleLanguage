# Citation Style Language plugin

An OJS/OMP/OPS plugin to generate an article citation in any CSL citation style using [citeproc-php](https://github.com/pkp/citeproc-php).

## CSL Files

The included CSL files are from the [official citation styles repository](https://github.com/citation-style-language/styles)(v1.0.1) developed by the [Citation Style Language](https://citationstyles.org/) project.
